## jlox

-   Compile:
    ```bash
    cd jlox/src
    javac com/craftinginterpreters/lox/Lox.java
    ```
-   Run REPL:
    ```bash
    java com/craftinginterpreters/lox/Lox
    ```
-   Run interpreter on file:
    ```bash
    java com/craftinginterpreters/lox/Lox demo.lox
    ```
